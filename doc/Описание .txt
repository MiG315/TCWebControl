Сайт для Rails

Функционал:

1) Загрузка в базу данных информации о результатах прохождения тестов производительности
2) Отображение через веб интерфейс

Структура:
Сайт сейчас представляет из себя 1 представление и 1 контроллер  - testjson.
Обмен данными происходит через json.
В качестве модели используется модуль datadb.rb из .\app\helpers. В его задачи входит парсинг тестов из 
сетевого хранилища, проверка этих данных на использование в БД и последующая их загрузка туда.
Этот же модуль отвественнен за выборку данных из БД. Данные выбираются за определенный период (начальная дата, конечная дата)
и для определенного теста.
БД - sqlite3. Модель активирует механизмы ссылочной целостности в sqlite3.


Джемы необходимые для работы :

gem install interface
